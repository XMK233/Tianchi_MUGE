电商图像描述生成
1. 任务描述
图像描述生成是一项经典的多模态任务，任务目标是根据一张图像生成相应的文字描述，要求生成的描述须如实反映图中的物体以及关键细节。电商领域有着众多的商品图片，将图像描述技术应用于电商领域，为每个商品生成一段富含吸引力的描述，对于吸引用户的点击、提高转化率有着重要的意义。

2. 任务说明
本项任务涵盖了服装、食品、化妆品、3C数码配件等众多商品类目，对于一件商品，我们会给出它的图片以及多条相应的商品描述作为训练数据，其中每张图片均以base64编码表示，参赛者需要根据商品图片生成对应的商品描述。标注数据的示例如下：

3. 评测指标
评测指标分为三项：

CIDEr-D：从tf-idf的角度考量候选描述与参考描述之间的相似度
BLEU-4：比较候选描述和参考描述里ngram的重叠程度，用于衡量n-gram的准确率
ROUGE-L：基于候选描述和参考描述的最长公共子序列（LCS）来考量候选描述的质量
其中CIDEr-D为主指标。在计算n-gram时，我们对中文进行分字处理，并将连续的字母或数字视作一个token。

4. 评测数据
本评测开放训练集数据50000张图片，验证集数据5000张图片，测试集10000张图片，其中每张图片对应多条描述（最多10条）。

数据集名称为：ECommerce-IC (E-commerce Image Caption Dataset)

目录为：/mnt/d/forCoding_data/Tianchi_MUGE/originalData/ECommerce-IC

文件解释如下：
IC_train.tsv：训练集对应的图片（base64编码）
IC_train.jsonl：训练集对应的caption
IC_valid.tsv：验证集对应的图片（base64编码）
IC_valid.jsonl：验证集对应的caption
IC_test.tsv：测试集对应的图片（base64编码）
IC_test.jsonl：需预测的文件，每条JSON数据仅给出img_id, 选手需要补充"text"字段。
example_gold.jsonl：标准答案示例
example_pred.jsonl：提交结果示例
README.txt：说明文件
